# KnowMe · 知我

**不再重复介绍自己——让每个 AI 在回答前先认识你。**

---

## 这是什么

KnowMe 是一个 Chrome 浏览器插件。

它解决一个很具体的问题：每次打开 AI 对话框，你都要重新介绍自己——你是谁、你在做什么、你的写作风格、你的项目背景……同样的话，说了一遍又一遍。

KnowMe 把这些背景信息存成"卡片"，打开任何 AI 网页时，一键注入到对话框，省掉那段反复铺垫的过程。

如果你同时使用多个 AI，插件还提供了一个整合工作台，把不同平台的回答放在一起，生成一条整合提示词，让 AI 帮你合成一份最终结果。

## 安装方法

KnowMe 目前是开发者模式安装，尚未上架 Chrome 应用商店。安装步骤如下：

**第一步：下载代码**

点击页面右上角绿色的 `Code` 按钮，选择 `Download ZIP`，下载后解压到你电脑上任意一个文件夹，比如桌面。

**第二步：打开 Chrome 扩展管理页面**

在 Chrome 地址栏输入：

```
chrome://extensions
```

回车进入扩展管理页面。

**第三步：开启开发者模式**

页面右上角有一个"开发者模式"的开关，打开它。

**第四步：加载插件**

点击左上角出现的"加载已解压的扩展程序"按钮，选择你刚才解压的文件夹，点击确认。

**第五步：确认安装成功**

插件列表里出现 KnowMe，说明安装成功。打开任意 AI 网页，页面右侧会出现 KnowMe 的侧边栏。

## 使用方法

### 背景卡片

1. 打开侧边栏，点击「背景卡片」tab
2. 填写卡片标题和内容，选择分类（工作 / 学术 / 其他），保存
3. 内容建议写完整一些，包括你的身份、工作语境、常见任务、偏好风格，越详细 AI 理解越准确
4. 你也可以把它当作提示词管理器，平时看到觉得不错的提示词，随手就做个小卡片
5. 打开任意 AI 网页，在卡片右侧点击「注入」按钮，背景信息自动填入对话框
6. 直接写你的问题发出去即可

> 💡 **小建议**：一句话的背景意义不大，建议把卡片写得丰富一些。比如不只是"我是运营"，而是"我在一家互联网公司做内容运营，负责公众号，用户群体是职场女性，领导喜欢结论前置的汇报风格……"

### 整合工作台

适合同时使用多个 AI、想把不同平台的回答合并成一份的场景。

1. 切换到「整合工作台」tab
2. 顶部选择平台（Claude / ChatGPT / Gemini / 豆包 / Kimi 等）
3. 把对应平台的 AI 回答粘贴进输入框
4. 在整合需求里描述你想要的结果，或者点击快捷标签选择预设需求
5. 点击「生成提示词」，复制生成的提示词
6. 粘贴到任意 AI 对话框，让 AI 帮你整合出最终版本

### 数据备份

卡片数据存储在浏览器本地，换电脑或重装浏览器前建议先导出备份。

在「背景卡片」页面底部的「数据备份」模块，点击「导出卡片」保存为 JSON 文件；换设备后点击「导入卡片」恢复。

## 常见问题

**Q：注入之后 AI 对话框没有内容？**

不同平台的输入框结构不同，部分平台可能需要点击一下输入框激活后再注入。如果多次无效，可以手动点击卡片的复制按钮，再粘贴到对话框。

**Q：数据会上传到服务器吗？**

不会。所有卡片数据完全存储在你自己浏览器的本地存储里，不经过任何服务器，不联网，不收集任何信息。

**Q：支持 Edge 浏览器吗？**

理论上支持，Edge 兼容 Chrome 插件，安装步骤相同。但目前主要在 Chrome 上测试，如有问题欢迎反馈。

**Q：会上架 Chrome 应用商店吗？**

计划中，目前还是早期版本，等功能稳定后会提交审核。
