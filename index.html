01 下载或克隆项目到本地

把整个项目文件夹保存到电脑上，确保你能看到 `manifest.json`、`popup.html`、`sidepanel.html` 这些文件。

02 打开 Chrome 扩展管理页面

在浏览器地址栏输入下面这行，然后回车进入扩展管理页。

chrome://extensions
03 打开右上角的“开发者模式”

开启后，页面左上角会出现“加载已解压的扩展程序”按钮，这就是本地安装入口。

04 选择当前项目文件夹完成加载

选中这个项目所在目录后，KnowMe 就会出现在你的扩展列表里。接着打开 ChatGPT、Claude 或其他支持平台即可开始使用。

Trust
最重要的不是“更炫”
而是你愿不愿意把自己的长期背景放进去

所以数据默认保存在浏览器本地，不经过服务器。你可以随时导出为 JSON 备份，也能在换设备时再导入回来。这个项目首先应该让人放心，其次才是让人惊艳。

  • 本地存储

    所有卡片内容保存在浏览器本地，不上传，不联网，不额外采集信息。

  • 导入导出

    支持 JSON 备份，迁移设备或重装浏览器之前，可以先把你的卡片安全带走。

    现在适合先快速验证产品价值，后续再决定是否扩展成正式官网或上架商店版本。
